### Tipos

#### Endpoints (CRUD basico)
---POKEMONES---
GET /api/v1/pokemons
GET /api/v1/pokemons/:id
POST /api/v1/pokemons/
PUT /api/v1/pokemons/:id
DELETE /api/v1/pokemons/:id

---TIPOS---
GET /api/v1/tipos
GET /api/v1/tipos/:id
POST /api/v1/tipos/
PUT /api/v1/tipos/:id
DELETE /api/v1/tipos/:id
